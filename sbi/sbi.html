This is a collection of useful functions shown in my teachings in the course Statistical Bioinformatics as well a few for the course Databases and Practical Programming using R in 2021/2022. Below you find the documentation for the following functions and operators:

Functions:

  • sbi$assoc - patch for assocplot to display residuals with color codes (plot).
  • sbi$bezier - create bezier lines using three coordinates
  • sbi$cohensD - effect size for the difference between two means (stats)
  • sbi$cohensF - effect size for Anova (stats)
  • sbi$cohensH - effect size for 2x2 contingency tables (stats)
  • sbi$cohensW - effect size for 2x2 and larger contingency tables (stats)
  • sbi$corplot - visualize a correlation with abline (plot)
  • sbi$corr - calculate all pairwise correlations for a given data frame or matrix (stats)
  • sbi$corrplot - visualize a matrix of pairwise correlations (plot)
  • sbi$cor.var - create data vector with a certain correlation to an other vector (stats)
  • sbi$cor.vars - create data frame with random values matching the given correlation matrix (stats)
  • sbi$chr2ord - convert factors or characters to ordinal numbers
  • sbi$cv - coefficient of variation (stats).
  • sbi$dict - create a key value list with unique keys (data)
  • sbi$dpairs - improved pairs plot with xyplot, boxplot or assocplot depending on the variable types (plot)
  • sbi$dpairs.legend - adding legends to pairs and dpairs plots (plot)
  • sbi$drop_na - remove all rows where any of the given columns contain a NA (data)
  • sbi$epsilonSquared - effect size measure for a Kruskal or Wilcox test (stats)
  • sbi$etaSquared - effect size measure for Anova or a linear model (stats)
  • sbi$file.cat - displays a file to the terminal (file)
  • sbi$file.head - displays the first lines of a file to the terminal (file)
  • sbi$flow - very simple flowcharter (plot)
  • sbi$fmt - Python like formatting of strings using curly braces (text)
  • sbi$gmean - geometric mean for a numerical vector, mean of products (stats)
  • sbi$hmean - harmonic mean for a numerical vector, mean of ratios (stat)
  • sbi$import - module like sourcing of R files relative to the main script (package)
  • sbi$is.dict - check if a list is a key-value list with unique keys (data)
  • sbi$kurtosis - fourth central moment of a distribution (stats)
  • sbi$lmPlot - xy-plot with linear model and the confidence intervals (plot)
  • sbi$mhist - lattice like histograms (plot).
  • sbi$mkdoc - literate programming for your R-code (tool)
  • sbi$mi - mutual information for two numerical variables or a binned table (stats)
  • sbi$modus - get the modus for a categorical variable (stats).
  • sbi$package.deps - return the dependencies of the given package (package)

  • sbi$pca.biplot - improved biplot for pca objects (plot)
  • sbi$pca.pairs - improved pairs plot for pca objects (plot)
  • sbi$pca.plot - improved screeplot for pca objects (plot)
  • sbi$pcor - calculate partial correlation for two variables corrected for one or more others (stats)
  • sbi$pcor.test - calculate partial correlation and test statistics for two variables corrected for one or more others (stats)
  • sbi$readDataFiles - read a directory of data files with the same function and settings (data).
  • sbi$report.pval - report a p-value using threshold or star syntax (stats)
  • sbi$shape - create polygon shapes centered at given x and y coordinates (plot)
  • sbi$shell - execute shell commands for embedding their ouput - Unix only (tool)
  • sbi$skewness - third central moment of a distribution (stats)
  • sbi$sem - calculate standard error of the mean (stats)

Usage

To use the functions just source the file sbi.R into your R application:

# notrun
source("sbi.R")

sbi$assoc(…,shade=TRUE)

Create assocplots with residual coloring.

This function updates the standard assocplot function from the graphics package with the ability to display residual colors. In blue and red are shown groups with residuals above +4 or below -4 in light colors are shown residuals between 2 and 4 for positive and -4 and -2 for negative residuals.

Arguments:

  • … - arguments delegated to the standard assocplot function.
  • shade - should the residuals been shown, default: TRUE

Examples:

  x <- margin.table(HairEyeColor, c(1, 2))
  sbi$assoc(x)

sbi$bezier(p1,p2,p3,…)

Create bezier lines using three coordinates.

This function creates bezier lines based on three coordinates. For more background information consult this discussion on stats.stackexchange.

Arguments:

  • p1 - starting point as vector of x and y
  • p2 - curve point as vector of x and y
  • p3 - end point as vector of x and y
  • plot - should the curve added to an existing plot, default: FALSE
  • arrow - should an error be plotted if plot=TRUE, default: FALSE
  • arrow.pos - where the place the arrow, between 0 and 1, default: 0.55
  • lwd - line width for line and arrow, default: 2 _ … - other arguments forwarded to the line and the arrows function.

Returns: if plot=FALSE returns list with x and y coordinates

Examples:

sbi$cohensD(num,cat,paired=FALSE)

Effect size for the difference between two means.

The function cohensD calculates the effect size for the difference between two means. Due to Cohen’s rule of thumb values of 0.2 to 0.5 are considered to stand for small effects, values from 0.5 to 0.8 represent medium effects and values above 0.8 represent large effects. Alternatively you could use the biserial correlation coefficient r as demonstrated in the example below.

Arguments:

  • num - vector with numerical values
  • cat - vector with two grouping variables, having the same length as num
  • paired - are the data paired, default: FALSE

Returns: Cohen’s d value

Examples:

  cohensD=sbi$cohensD
  set.seed(125)
  data(sleep)
  with(sleep,cohensD(extra,group))
## [1] 0.8321811
  x1=rnorm(100,mean=20,sd=1)
  x2=rnorm(100,mean=22,sd=1)
  g1=rep('A',100)
  g2=rep('B',100)
  # difference should be around 2SD
  cohensD(c(x1,x2),as.factor(c(g1,g2)))
## [1] 1.898655
  # biseriell correlation coefficient as alternative
  # value is as well large
  cor(c(x1,x2),as.numeric(as.factor(c(g1,g2))))
## [1] 0.6903109

See also: sbi$cohensW

sbi$cohensF(x,g)

Effect size for an Anova.

The function cohensF calculates the effect size for two variables in an Anova. Due to Cohen’s rule of thumb values of 0.1 to 0.25 are considered to stand for small effects, values from 0.25 to 0.4 represent medium effects and values above 0.4 represent large effects.

Arguments:

  • x a numercial vector
  • g a factor variable with at least three levels

Returns: Cohen’s f value

Examples:

  data(iris)
  sbi$cohensF(iris$Sepal.Width, iris$Species)
## [1] 0.8178289
  sbi$cohensF(iris$Sepal.Length, iris$Species)
## [1] 1.273831

See also: sbi$cohensD, sbi$etaSquared

sbi$dpairs(data,…)

Improved pairs plot considering the data types.

The function dpairs provides an improved pairs plot which accounts for the data type of the actual variables. It will plot in the lower diagonal xy-plots, box-plots or assoc-plots depending on the two data types. In the upper diagonal effect sizes and stars for the p-values for the tests (anova, t.test, chisq.test or cor.test will be shown. In the diagonal the data distribution will be outlined. This plot is usually an useful visualization for 3-8 variables.

Arguments:

  • data - data frame with columns of class factor, numeric or integer.
  • col.box - colors for the boxplots, either a single value or a vector of colors for each level of a factor variable, default; ‘grey80’
  • col.xy - colors for the xy-plots, either a single value of a vector which is as long as the number of data points, default: ‘grey60’
  • cex.diag - character expansion for the diagonal texts
  • order - should the variables be ordered by data type and name, this is recommended as it orders the plots, starting with assocplots, then boxplots and finally xyplots, default: TRUE
  • pch - plotting character for xy-plots, default 19 (round circle).

Returns: numerical value for the excess kurtosis.

Example:

  data(iris)
  dpairs=sbi$dpairs
  par(omi = c(0.8, 0.4,0.4,0.4))
  dpairs(iris,col.box=2:4,col.xy=rep(c(2:4),each=50),
     cex.diag=1.6)
## The following objects are masked _by_ .GlobalEnv:
## 
##     cohensD, cv, dpairs
